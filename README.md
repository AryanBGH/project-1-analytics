# Project 1 — Analytics to Decision

## Objective
This project focuses on translating raw data into clear,
decision-oriented insights through structured analysis.

## Scope
- Data understanding and exploratory analysis
- Insight generation and decision framing
- No model training in this project

## Constraints
- Python execution is not available in the local environment
- Analysis is performed using non-Python tools (spreadsheets or SQL)
- All results are documented transparently without false execution claims

## Current Phase
Week 2: Problem framing and dataset selection
## Dataset

The dataset used in this project is the **Customer Churn Dataset** sourced from Kaggle.

- Source: https://www.kaggle.com/datasets/muhammadshahidazeem/customer-churn-dataset
- File used: `customer_churn_dataset-testing-master.csv`
- License: GPL-2

The dataset is stored in the repository under:data/raw/customer_churn_dataset-testing-master.csv
The raw dataset is kept unchanged to preserve traceability.
